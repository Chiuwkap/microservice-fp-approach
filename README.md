## Microservices.fp.approach

Simple guide to start:

1. git clone URL
2. cd into the repo
3. gradle bootRun

Microservice & Funcational Programming style approach

- JDK 11
- Java 8 Streams API
- Lombok - plugin might needed in IDE
- Immutable Objects
- Gradle 6.7
- Rest API
- Swagger 
- Postman or Swagger-UI
- H2 Database
